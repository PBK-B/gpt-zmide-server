# gpt-zmide-server 💡

> zmide ChatGPT 应用服务，用于管理应用程序对接和集成 ChatGPT API 的服务应用，提供简单易用的 API 服务。

## 开始 🎀

1. 创建 `app.conf` 配置文件

2. 启动服务 `go run .`

3. 访问 `http://127.0.0.1:8091`

## 文档 📜

API 文档请参考: [docs/README.md](/docs/README.md)
