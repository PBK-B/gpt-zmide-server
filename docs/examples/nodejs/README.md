# Typescript example of using gpt-zmide-server

## Start

edit the `token` field in the `const server = new GPTServer("xxxxxx")` of the main.js file

```shell
# build lib
npm run build:libs

# npm install or yarn
npm install

# use node run main.js
node main.js
```
